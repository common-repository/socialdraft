== Description ==

The Socialdraft plugin allows you to monitor and analyse reviews on your Wordpress site. Visit http://socialdraft.com/about/ for more info.

PS: You'll need a Socialdraft.com API ID and Key by signing up at http://socialdraft.com in order to use it.

== Installation ==

Upload the Socialdraft plugin to your blog, Activate it, then enter your Socialdraft.com API ID and key together with your email used.

1, 2, 3: You're done!
